 7. ADMIN INTEGRATION ✅
───────────────────────────────────────────────────────────────────────────────

Admin Dashboard → Client Portal Flow:

1. Admin updates listing via admin dashboard:
   - Edits images (gallery_urls, cover_image_url)
   - Updates specs (beds, baths, area, price)
   - Changes AI score
   - Updates payment plan, virtual tour URL
   - Toggles status (draft/active/sold)

2. Firestore receives the update immediately

3. Client component refetches data:
   - useEffect monitors for changes
   - fetchActiveListingById() pulls latest
   - UI re-renders with new data

4. Public sees changes instantly:
   - No cache invalidation needed
   - No re-deployment required
   - Real-time sync through Firestore

Real-Time Sync Mechanism:
  • Component uses useEffect(() => { fetchData() }, [id])
  • Re-fetch triggers on component mount
  • Firestore listeners (built into SDK) auto-sync
  • No polling required
