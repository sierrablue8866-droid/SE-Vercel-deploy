# FAQ

### Does the Seed require an internet connection?
No. The Seed is fully functional offline. Cloud sync is optional.

### What data leaves the device?
Nothing, by default. If you opt in to cloud sync, only encrypted telemetry is sent.

### Can I build my own cogs?
Yes - a cog development guide is coming soon.

### How do I reset the Seed to factory defaults?
Unplug, hold the reset button (if present on your model), and replug. Or use the `/api/v1/factory-reset` endpoint.

### Is the firmware open source?
The firmware source is currently closed. Documentation and community tools are open.
